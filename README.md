### Install
> requirements: node>=20

## get-started
```
# cd to your project directory
cd ~/projects/applyai-frontend
nvm use

# install dependencies
yarn

# 找同事要一个 .env.development 文件(同事那边是最新的) 放到项目 config/envs 文件夹下
yarn dev
```

visit http://127.0.0.1:5174/
